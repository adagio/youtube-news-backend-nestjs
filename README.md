# Backend for Youtube News app

## History

Check out our [migration story](HISTORY.md) from Python FastAPI to TypeScript NestJS, powered by AI pair programming.

## Running the Project

### Prerequisites
- NodeJS v22
- Docker (optional)

### Change current directory to 'youtube-news'
```sh
cd youtube-news
```

### Setup
```sh
npm install
```
### Run
```sh
npm run start:dev
```
